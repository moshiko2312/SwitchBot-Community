# SwitchBot Community

## ConX Keypad Bridge

מדריך התקנה והפעלה מלא עבור ConX Keypad Bridge ו‑SwitchBot Keypad Vision.

### תכונות

- חיבור ל‑Home Assistant
- תמיכה ב‑PIN, טביעת אצבע וזיהוי פנים
- Guest Passcodes
- אינטגרציה עם SwitchBot Cloud
- אוטומציות מוכנות ל‑Home Assistant

### מדריך מלא

ראה את הקובץ:

- `SwitchBot-Keypad-Bridge-Guide.md`

### אוטומציות לדוגמה

#### Face Recognition
```yaml
state: face
```

#### Fingerprint
```yaml
state: fingerprint
```

#### PIN
```yaml
state: pin
```
